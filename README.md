## small SNES-like mosaic effect for [jQuery](http://jquery.com)
This small jQuery plugin simply animates the specified image file with a mosaic effect.

![Mosaic Example](img/mario.gif)

## Usage

    jQuery.('#myImage').mosaic({
        repeat:true
    });
